导出文件命名规则

一、作用与功能

1. 所有文件导出格式定义的相应编号规则均在此定义；

2. 不同的文件导出格式可以使用相同的导出文件命名规则；

二、自动编号规则设计原理

为便于用户理解，我们先来了解一下编号基本的设计原理：

号码若干段值+序号组成。即：前缀+{字段值+本段分隔符}n+序号。段之间可以有分隔符。

前缀是一个固定值，段值实际就是单据上的字段值，当各段值都相同时，序号递增1产生一个新号。如采购订单的编号规则是：

PO+[采购单位]+[日期]+序号，则‘0001’单位在2002年6月28日发送了第一个订单。则这个编号可以编为：PO-0001-20020628-0001。

从上面可以看出，关键在于字段的编码规则定义。部件中所有字段都可用来编号，用户根据需求自行选择其中若干字段。用户要给定其排列顺序，并指明起始号是多少。

另外，当规则改变时，在某一编号中，新的规则中与旧规则重复，那么系统则在以前的基础上编号。比如：

PO+[采购单位]+序号，有一单位代码是’2002’，已经编到20号。当前单据是PO-2002-0020。

现将规则改为PO+[年]+序号，而这年刚是2002年，这时，在新的规则与旧规则各段是重复的。所以新规则不能从1开始，而是在以前的基础上开始编。所以新规则的第一个单据号是PO-2002-0021。

三、操作说明

系统以列表方式显示所有已定义的自动编号规则信息，单击某一记录可打开“自动编号规则定义”编辑窗口，用户可对原信息进行修改，单击“新增”可增加新一条记录。在此定义的自动编号规则可在文件导出格式定义中调用。

1. 新增自动编号规则定义

移动鼠标点击窗口上方“新增”键，系统弹出“自动编号规则”编辑窗口，可对编号规则进行定义；

2. 编辑自动编号规则

编辑窗口分上下两部分，上部分主要用于定义某编码规则的基本显示信息，下部分主要用于对该种编码规则所包含自动编号字段的各相应编号规则的定义，我们分别加以说明。

 

（1） 编码规则基本信息

★        编码规则：即是系统区别编码规则的唯一标志，一般为数字或字母，最长为30个字符，唯一且不能为空；

★        编码规则名称：设置编码规则显示名称，一般为汉字或字母，最长为60个字符，不能为空；

★        编号前缀：设置编号规则的前缀，是一个固定值，可以为字母、汉字、数字及其他符号，最长为10个字符；

★        编号顺序段长度：设置自动编号顺序号的长度，为0～9的数字，系统自动以0补足空缺位置；

★        是否连续编号：该选项表示当前缀不同时，其编号顺序是否连续，选择“是”则不考虑前缀的变化，选择“否”则根据前缀的变化自动更新编号顺序。

 

（2） 字段编号规则定义

点击窗口下方“新增”键，则系统最末一行自动增加一行记录栏，可对以下内容进行设置：

 

★        排序号：即对所选的字段进行编号前后顺序的排序，排序在前的字段先参与编号，排序在后的后参与，由系统自动给出，同时允许用户修改。

★        参与编号字段：点击该行出现的选择按钮，在弹出列表框中选择需要参与编号的字段；

★        本段分隔符：可对所选字段添加分隔符，这样就可将该段与其他字段区分开来，系统提供“减号”、“下划线”、“斜杠”、“反斜杠”、“空格字符”五种符号形式供用户选择。

★        段固定长度：为0～9的数字。设置段固定长度是因为参与编号的字段值可能长度不同，为了保证编号的一致性，可以在编号规则中将参与编号的字段设为固定长度。这样如果字段值的长度不足，可以用规定的字符在适当位置将其补足。

★        补空位置：如设置参与编号的字段为固定长度，则需要在此设置其适当的补空位置，系统提供“补前位”、“补后位”两种补空位置供用户选择。

★        补空位字符：如设置了补空位置，则还需要设置相应的补空字符，系统提供“零”、“空格字符”两种补空位字符供用户选择。

★        日期段格式：如该字段需要包含日期，则在此可对日期型字段的显示格式进行定义，点击该单元格出现的下拉按钮，系统提供YYYYMMDD，YYMMDD，YYYY，YYYYMM，YYMM五种格式供用户选择。

 

举例说明：

如上例编号为：PO-0001-20020628-0001，其中“PO”为编号前缀，“-”为分隔符，“0001”为采购单位字段，“20020628”为日期，设置其日期格式为“YYYYMMDD”，“0001”为递增的顺序号，设置其编号顺序长度为4位。

3. 保存、修改

所有信息定义完毕后点击窗口“保存”键，对本次操作确认存盘，如需要继续增加，可点击窗口上方“新增”键，在不退出当前编辑窗口的情况下继续增加新信息，点击“修改”键，用户可重新定义除编码规则以外的所有信息，修改完毕，点击“保存”确认存盘。点击“退出”键则关闭当前操作窗口，返回上一级页面。
